`int pow(int x, int n)` calculates the power of a number

`int fact(int n)` calculates the factorial of a number

`int comb(int n, int k)` calculates the combination of n choose k

`int perm(int n, int k)` calculates the permutation of n choose k

`int inf()` returns the positive infinity

`int ninf()` returns the negative infinity

`int max(int x, int y)` returns the maximum of two numbers

`int min(int x, int y)` returns the minimum of two numbers

`int abs(int x)` returns the absolute value of a number

`int gcd(int a, int b)` calculates the greatest common divisor of two numbers

`int lcm(int a, int b)` calculates the least common multiple of two numbers

`pair(int, int) _preRoundedLog(int n, int base)` helper for rounded log functions, performs ceiled log but also returns the exponented value to allow further manipulation

`int ceiledLog(int n, int base)` log and then ceil the result

`int flooredLog(int x, int n)` log and then floor the result
